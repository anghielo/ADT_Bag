# ADT_Bag
Abstract Data Structure - Bag

Purpose:
Masterthe interface and implementations of bags.

TaskDescription:
Task1: Add three  methods: union, intersection,  and difference to the interface BagInterface for the ADT bag.
Task2: Implement the three methods,union, intersection, and difference,for the class ResizeableArrayBag.
Task3: Implement the three methods,union, intersection, and difference,for the class LinkedBag.
Task 4: Write a client program, “ArrayBagTest.java”, which contains a main method to test  the  three  methods (union, intersection,  and difference) you implemented for the  class ResizeableArrayBag.
Task 5: Write a client program, “LinkedBagTest.java”, which contains a main method to test  the  three  methods (union, intersection,  and difference) you implemented for the  class LinkedBag.
